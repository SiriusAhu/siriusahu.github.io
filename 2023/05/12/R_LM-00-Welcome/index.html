What is Linux Mint

Linux Mint是一个基于Ubuntu的Linux发行版。（Linux Mint Official Website）

发行版（distribution）是指Linux内核与应用软件的集合，通常包含了Linux内核、GNU工具与图形桌面环境，以及大量应用软件。

Why Linux Mint

Friendly to Newbies

Linux Mint是一个非常适合新手的Linux发行版，它提供了多种桌面环境供用户选择，其中Cinnamon桌面环境是一个非常类似于Windows的桌面环境，对于Windows用户来说，上手非常容易。

其官网上有着这样一句话：

Give it a try. You’ll Love it!
Most of our users come from Windows and they never look back.

(不过我们不一定非要“never look back”，毕竟有一些软件、游戏是Windows独有的，在需要的时候还是可以回到Windows的。只是如果你不体验一下Linux，你就永远不知道Linux有多好。)

Stable and Powerful Community

我想用Arch和Linux Mint来做一个比较。

虽然Arch（另一个Linux发行版）有着公认的最强大的wiki指南和一群十分厉害的用户组成的社区，你可以依靠它们解决你遇到的几乎任何问题，但是，解决问题也需要时间。

“Yeah, I think Arch is best if you want to run all the latest packages and very experienced with Linux. But I only use Linux for work, so I need it to be stable and usable. Though I can fix most problems with the support of Arch wiki and their community, it takes time. So I finally chose Linux Mint, which is both great for experienced users and new users.”
——这是我一位使用了十年以上Linux Mint的朋友对我说的话。

诚然，Arch的wiki和社区是非常强大的，但是，在你遇到问题的时候，你仍需要花费许多时间去解决它们，更何况Arch的入门门槛也比较高。而Linux Mint的社区也是非常强大的，并且由于它基于Ubuntu，所以你也可以在Ubuntu的社区寻找解决方案。（不过这并不是完全可行的，因为Linux Mint和Ubuntu的一些配置是不同的。）

Easy to Install

Linux Mint提供了图形化的安装界面，你只需要简单地按照提示进行操作，就可以轻松地安装Linux Mint。

Welcome Screen

Linux Mint也提供了一个welcome screen窗口，用户可以通过它的引导来配置很多的基础功能（包括驱动、系统备份等）

Summary

总之，Linux Mint是一个非常适合新手的Linux发行版，希望你能在使用它的过程中感受Linux的魅力。

Have fun! :wink:
